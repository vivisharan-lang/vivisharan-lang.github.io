☁️ Cloud Network Monitoring API

🧠 What is this project?

This is a cloud-based backend API that simulates a telecom network monitoring system. It provides endpoints to check system health and network traffic.

⚙️ What does it do?

  • Returns network status
  • Shows system health
  • Simulates traffic load

🌍 Live Demo

Open API

Try:
/ → status
/health → health check
/traffic → traffic

🛠️ How I built it

1. Created a backend API
Used Python + Flask and built endpoints for status, health, and traffic.

2. Containerized the app
Used Docker and created a Dockerfile to package the application.

3. Deployed to Google Cloud
Used Cloud Run and Artifact Registry to deploy and host the app.

4. Debugged deployment issues
Fixed IAM permission errors and managed service accounts.

🧠 What I learned

  • How cloud services run backend applications
  • How Docker containers work
  • How to deploy services using Cloud Run
  • How to debug real cloud permission issues

💬 Why this is relevant

This project simulates how telecom systems expose APIs for monitoring network health and traffic in cloud environments.
