🌍 Travel Memory Map

🧠 What is this?

An interactive map where users can store travel memories by placing pins on locations around the world.

⚙️ Features

  • Add pins anywhere on the map
  • Attach text and images
  • Save data using browser storage

🛠️ Tech

JavaScript, Leaflet, LocalStorage
